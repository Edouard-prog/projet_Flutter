# Mini WhatsApp Flutter

Un mini clone de WhatsApp pour apprendre Flutter.

## Installation
1. Cloner le dépôt : `git clone https://github.com/Edouard-prog/ProjetFlutter.git`
2. Ouvrir le projet dans Android Studio ou VS Code
3. Lancer : `flutter pub get` puis `flutter run`

## Fonctionnalités
- Envoyer et recevoir des messages
- Liste des conversations
- Navigation avec BottomNavigationBar
